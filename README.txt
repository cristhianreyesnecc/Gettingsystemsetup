ACME CORP

Read this before install

This is a guide to install Cent OS and Ubuntu servers on a Virtual machine. Select the server you want to install and follow along with the steps. There are five files in this folder ReadMe.txt, UbuntuServer.docx, CentOSServer.docx, and Nettools.docx and Intro,docx. The guides will show you how to do the install. Do not proceed with install if you do not have any of the following:
* VMware Workstation
* Laptop
